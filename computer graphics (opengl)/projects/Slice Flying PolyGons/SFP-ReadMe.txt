실행시 유의 사항
- 전처리 문에서 glm같은 경우 저의 파일 위치 때문에 include <glm/...>형식으로 되어있습니다. 그래서 혹시 오류가 발생하면 경로를 직접 바꿔주세요...
- 제 컴퓨터에선 이상하게 64비트에서만 돌아가는데 혹시 참고 해주세요.
- 배경음악도 구현해서 해당 파일을 같이 위치 시켜주세요.

명령어 목록
- T/t: 경로 표시
- E: 강제 엔딩
- q/esc: 강제 종료
- P/p: 멈춤(슬라이스,셀 이동은 가능)
- +/-: 속도
- L/l: '라인/필' 그리기
- B/b: 바운딩 박스 보기(밑에 셀과 충돌 검사를 위한 바운딩박스)
- 도형들 움직이는 공간 마우스 왼쪽 클릭: 슬라이스 시작 / 마우스 왼쪽 버튼 올림: 슬라이스 끝(여기서 슬라이스 검사)
- 셀(아래 3층칸) 왼쪽 클릭 & 드래그: 셀내 도형 위치 옮기기

추가 구현
- 배경음악 추가
- 두번 이상 자르기
- 셀(아래 3층칸) 채워질 시 색깔 빨강으로 바뀜
- 셀 채워질 시 속도 증가
- 엔딩 구현
- 도형 셀 크기와 자동 맞춤 (충돌시)
- 멈춤 기능
- 슬라이스 길이 제한
- 슬라이스 성공시 색깔 빨강으로 변화
